# AI Tutor API Documentation

This directory contains the versioned API structure for the AI Tutor course generation system.

## API Versions

### v1 (Current Stable Version)
- Standard course generation functionality
- Module and lesson content creation
- Quiz generation
- Basic course export (JSON only)

### v2 (In Development)
- Enhanced course generation with more customization options
- Support for additional export formats (PDF, DOCX)
- User management and course sharing
- Analytics and progress tracking
